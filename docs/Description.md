# ✈️ 耀坤飞控 · 开源四翼无人机

基于 Arduino 的高度模块化飞控 | 专为嵌入式学习 · 算法验证 · 科创竞赛而生

## 项目简介

**旗舰项目：🚁 耀坤飞控 (YaoKun Drone)**

深度模块化的 Arduino 飞控，完整支持 Madgwick/Mahony/互补滤波三种姿态解算器，串级 PID 控制 + WebSocket 网页直连遥控。代码拆分为 12 个功能文件，中文注释详尽，零门槛上手，是嵌入式系统教学的完美案例，也是科创竞赛和算法快速验证的理想平台。

**对比开源项目**

- **Flix** – 极简 ESP32 四轴，<2000 行 Arduino 代码，支持 MAVLink + Gazebo 仿真，定位教育和科研
- **ESP-FC** – 模拟 Betaflight 4.2 的高性能飞控，支持 Dshot 等高级 ESC 协议，黑盒记录，适合穿越机玩家
- **ESP-Drone** – 乐鑫官方 FreeRTOS 无人机，继承 Crazyflie 算法，支持定高/定点，提供 Android/iOS APP 控制

## 🔧 一、核心架构与系统对比

| 维度 | 耀坤飞控 | Flix | ESP-FC | ESP-Drone |
|------|----------|------|--------|------------|
| 开发框架 | Arduino（模块化拆分12个文件） | Arduino（单体结构，<2000 行） | 原生 ESP-IDF / Arduino 混合 | ESP-IDF + FreeRTOS |
| 控制循环频率 | 500 Hz | 1000 Hz | 1-2 kHz（可配置） | 500 Hz |
| 操作系统 | 裸机 | 裸机 | 裸机 | FreeRTOS 实时操作系统 |
| 协议支持 | WebSocket + HTTP | MAVLink（QGC 兼容） | Betaflight Configurator 兼容 | CRTP + cfclient / APP |
| 参数配置方式 | 网页端直连调参（无需额外App） | MAVLink 参数服务 | Betaflight 配置器（CLI） | cfclient 上位机 / 手机 App |

> **架构解读：** ESP-Drone 基于 FreeRTOS，适合专业嵌入式开发；Flix 以极简著称，是绝佳的 Arduino 入门教材；ESP-FC 高度模拟 Betaflight，适合穿越机进阶；**耀坤飞控**则在 Arduino 框架下实现了深度的模块化拆分，代码结构清晰，兼具易读性和可维护性，在同类 Arduino 飞控中极为罕见。

## 🧠 二、姿态解算 & 传感器融合

| 算法维度 | 耀坤飞控 | Flix | ESP-FC | ESP-Drone |
|----------|----------|------|--------|------------|
| 姿态解算器 | **Madgwick / Mahony / 互补滤波**（3种可选，支持热切换） | 互补滤波 | 四元数 + 互补滤波 | Crazyflie EKF（扩展卡尔曼滤波） |
| 滤波器种类 | 低通 + 加速度计信任度自适应 | LPF 通用低通滤波 | LPF、动态 Notch、dTerm、RPM 滤波 | 标准传感器滤波 |
| 陀螺仪支持 | MPU6050 / MPU6500（自动检测，I2C+SPI） | MPU9250 / MPU6500 / MPU6050 | MPU6050, MPU9250, ICM20602, BMI160 | MPU6050 |
| 自适应算法 | 加速度计信任度动态调整 Beta | ❌ | 动态 Notch 滤波 | ❌ |
| 滤波器调参 | Web 界面实时调整 | CLI | Betaflight 配置器界面 | cfclient |

> **算法解读：** ESP-FC 支持高级滤波，适合追求极致性能的穿越机；ESP-Drone 继承了 Crazyflie EKF，专业但复杂；Flix 保持简单，适合教学入门。**耀坤飞控**在 Arduino 环境下实现了三种姿态解算器可选 + 自适应滤波，这种灵活性极具教学价值，可直观对比不同滤波器的效果。

## ⚙️ 三、PID 控制架构 & 飞行性能

| 控制维度 | 耀坤飞控 | Flix | ESP-FC | ESP-Drone |
|----------|----------|------|--------|------------|
| 控制架构 | 串级 PID（角度环 → 角速率环） | 三维二级串级 PID | Betaflight 风格 PID | Crazyflie PID |
| 偏航控制 | 独立偏航角速率 PID | 集成在三维 PID 中 | 标准 PID | 标准 PID |
| 扭矩缩放/抗饱和 | ✅ 低油门扭矩动态缩放 + 积分抗饱和 | ❌ | Betaflight 标准 | 标准 |
| 积分抗饱和 | ✅ 输出饱和时积分衰减 | ✅ | ✅ | ✅ |
| 飞行模式 | 自稳模式（可扩展定高） | ACRO / ANGLE（通过 MAVLink） | ACRO / ANGLE / AIRMODE | 自稳定 / 定高 / 定点 |

> **控制解读：** ESP-Drone 定高/定点功能最全面；ESP-FC 飞行模式丰富，适合穿越机竞技；Flix 保持核心自稳功能；**耀坤飞控**基于 Arduino 完整实现串级 PID，并加入扭矩缩放抗饱和，以最低门槛复现专业飞控核心算法。

- **🏆 深度模块化教学典范** – 拆分为 `rc.ino`、`control.ino`、`failsafe.ino` 等12个文件，每个文件职责单一，完美诠释嵌入式工程架构。
